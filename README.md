# 📝 To-Do List

Une application web simple et responsive pour gérer ses tâches du quotidien, installable sur mobile comme une vraie application.

## Fonctionnalités

- Ajouter une tâche
- Cocher une tâche comme complétée
- Supprimer une tâche
- Sauvegarde automatique via le localStorage (les tâches persistent après fermeture du navigateur)
- Chaque appareil possède sa propre liste, indépendante des autres

## Technologies utilisées

- HTML
- CSS (Flexbox, responsive design)
- JavaScript (DOM, localStorage)

## Installation sur mobile

1. Ouvre l'URL de l'application dans ton navigateur
2. **iPhone** → bouton partage → "Sur l'écran d'accueil"
3. **Android** → menu ⋮ → "Ajouter à l'écran d'accueil"
